# CLI

```
Usage: atprs-cli [OPTIONS] <COMMAND>

Commands:
  get-profile
  get-session
  get-timeline
  help           Print this message or the help of the given subcommand(s)

Options:
  -p, --pds-host <PDS_HOST>  [default: https://bsky.social]
  -c, --config <CONFIG>      [default: config.toml]
  -h, --help                 Print help
```
